#campfireMod HOW TO INSTALL

0. 将campfire文件夹放入newdts的modules文件夹中；
1. 打开Install文件夹，将sql文件夹内的文件放入对应目录；
2. 在modules.list.php文件中写入CampfireModuleslist文件中提供的模块信息；
3. 重新安装游戏，并重设模块缓存；
4. 冲！
